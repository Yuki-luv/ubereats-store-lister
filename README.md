# UberEats 店舗リスト取得ツール 📞

コールセンター業務向け — 指定地域の Uber Eats 加盟店情報をリスト化するツール

## セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. Playwright のブラウザインストール

```bash
playwright install chromium
```

### 3. アプリの起動

```bash
streamlit run app.py
```

## 使い方

1. 地域名を入力（例: 「新宿区」「渋谷区」「大阪市」など）
2. 取得件数を設定（デフォルト30件）
3. 「検索開始」ボタンをクリック
4. 結果テーブルで確認（電話番号は自動正規化 + 元データも表示）
5. CSVダウンロードで保存

## 機能

- 🔍 地域名による店舗検索
- 📞 電話番号の自動正規化（国番号→国内番号、ハイフン挿入）
- 📍 住所の正規化（不要な国名除去、全角半角統一）
- ⚠️ データ品質の自動チェック（桁数不足・形式不明をハイライト）
- 📊 元データと正規化データの並行表示（目視確認用）
- 📥 CSV エクスポート
- 🎯 コールセンター向け電話番号リンク（tel:プロトコル対応）

## 注意事項

- 電話番号・住所は自動正規化されていますが、必ず元データと照合してください
- 大量のリクエストは Uber Eats サーバーに負荷がかかるため控えめに
- サイト構造の変更により動作しなくなる可能性があります
